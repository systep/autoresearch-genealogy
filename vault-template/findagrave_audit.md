### Search Methodology Notes (2026-04-18)

1. **Find a Grave direct search blocked**: The findagrave.com memorial search API returns HTTP 403 on all programmatic access attempts. All searches were conducted via web search engine indexing of Find a Grave pages.
2. **Lincoln Park Cemetery coverage gap**: Despite having 13,809 memorials on Find a Grave, the Markel and Salk families appear to have minimal coverage. Kate Markel Kaplan (#232002112) is the only Markel; no Salk memorials were found. This is likely a coverage gap, not evidence of absence, since Abraham Salk's death certificate (Tier 1) confirms Lincoln Park Cemetery burial.
3. **High Street Cemetery coverage gap**: 1,372 memorials indexed, but no Bright family memorials found. Both Samuel Graham Bright (d. 1909) and Willis Henrietta Youngblood Bright (d. 1952) are documented as buried there.
4. **Southern family cemeteries**: Jeddiah Clark and George Bright Sr. died in rural areas (Simpsonville SC, Floyd County VA) in the 1860s-1870s. Many burials from this era are in small family cemeteries with no Find a Grave coverage.
5. **Name variant exhaustion**: For each person, 2-4 name variants were searched (e.g., Morris/Moishe, Willis/Willie, Tauba/Tobey/Toby, Helen Markel/Helen Brandt). No variants produced results.
